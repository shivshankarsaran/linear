Banach Spaces

1. Introduction

Banach spaces are complete normed vector spaces that provide a general framework for functional analysis. Completeness ensures sequences arising from approximation processes converge within the space. Their structure is fundamental for studying bounded linear operators and they are central in areas such as differential equations, optimization, quantum theory, and numerical analysis.

Completeness is what makes approximation methods and operator-theoretic tools reliable in infinite-dimensions.

2. Preliminaries

2.1 Normed Vector Spaces

Definition. A normed vector space (X, ‖·‖) consists of a vector space over ℝ or ℂ together with a function ‖·‖ : X → [0,∞) satisfying:

  1. ‖x‖ = 0 iff x = 0.
  2. ‖αx‖ = |α| ‖x‖ for scalars α.
  3. Triangle inequality: ‖x + y‖ ≤ ‖x‖ + ‖y‖.

2.2 Completeness

Definition. A sequence (x_n) in a normed space is called Cauchy if for every ε > 0 there exists N such that

‖x_n - x_m‖ < ε for all m,n ≥ N.

A normed space is complete if every Cauchy sequence converges to some element of the space.

3. Banach Spaces

Definition. A Banach space is a normed vector space that is complete; every Cauchy sequence converges to an element of the space. Completeness ensures approximation methods yield valid limits and that functional analysis tools remain applicable.

3.1 Fundamental Properties

  • Every closed subspace of a Banach space is a Banach space.
  • The continuous dual X* of any Banach space is itself a Banach space (with the operator norm).
  • Finite-dimensional normed spaces are automatically complete (hence Banach).
  • Products and quotients of Banach spaces preserve completeness under the usual constructions.

4. Examples

4.1 Sequence Spaces

For 1 ≤ p ≤ ∞, the space

ℓp = { x = (x_n) : ∑n=1∞ |x_n|⁽ᵖ⁾ < ∞ }

equipped with the usual p-norm is a Banach space.

4.2 Function Spaces

The space C([a,b]) of continuous real-valued functions on a closed interval with the sup norm ‖f‖∞ = supx∈[a,b] |f(x)| is a Banach space.

The Lebesgue spaces Lp(Ω) for 1 ≤ p ≤ ∞ are Banach spaces with respect to the p-norm as well.

5. Major Theorems

5.1 Hahn–Banach Theorem

Theorem (Hahn–Banach). Let M be a subspace of a normed space X and let f : M → ℝ be a bounded linear functional. Then there exists an extension F : X → ℝ such that ‖F‖ = ‖f‖.

5.2 Uniform Boundedness Principle

Theorem (Banach–Steinhaus). If a family of bounded linear operators from a Banach space to a normed space is pointwise bounded, then it is uniformly bounded.

5.3 Open Mapping Theorem

Theorem. If T : X → Y is a surjective bounded linear operator between Banach spaces, then T maps open sets in X onto open sets in Y.

5.4 Closed Graph Theorem

Theorem. A linear operator between Banach spaces is bounded if and only if its graph is closed in the product space.

6. Conclusion

Banach spaces provide a rigorous setting for modern analysis and operator theory. Their completeness, structural properties, and diverse examples make them indispensable in theoretical and applied mathematics.
